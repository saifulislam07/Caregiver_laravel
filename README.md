# Caregiver Service Platform (Laravel 12)

## 📌 Project Overview
This project is a modern caregiver service booking platform inspired by caregiveragencybd.com. It allows users to book caregivers, nurses, and home healthcare services online.

---

## 🛠️ Tech Stack

- Laravel 12
- Bootstrap 5
- jQuery & AJAX
- MySQL
- REST API

---

## ✨ Features

### 👤 User Features
- User Registration & Login
- Book Caregiver / Nurse Online
- View Services & Packages
- Track Booking Status
- Submit Reviews

### 🧑‍⚕️ Caregiver Features
- Profile with skills, experience
- Availability status
- Ratings & reviews

### 🛠️ Admin Panel
- Manage Users
- Manage Caregivers
- Manage Services
- Manage Bookings
- Manage Packages
- View Reports

---

## 📦 Modules

### 1. Service Module
- Create, update, delete services
- Service listing page

### 2. Booking Module
- Book service using AJAX
- Assign caregiver
- Status: pending, approved, completed

### 3. Caregiver Module
- Add caregiver
- Assign services
- Availability toggle

### 4. Package Module
- Pricing plans (Basic, Premium)
- Feature list (JSON)

---

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|--------|-------------|
| GET | /services | List services |
| POST | /booking | Create booking |
| GET | /caregivers | List caregivers |
| POST | /contact | Contact form |

## ⚡ AJAX Features

- Booking without page reload
- Dynamic caregiver availability
- Live search/filter
- Form validation
